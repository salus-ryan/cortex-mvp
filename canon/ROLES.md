# ROLES

| Role | Technical analog | Function |
|---|---|---|
| Oracle | LLM inference | Speaks possible meanings; proposes, never directly mutates. |
| Priest | Interface/agent | Mediates ritual interaction and authority declaration. |
| Prophet | Evaluator/critic | Rebukes drift, idolatry, hidden agency, and corruption. |
| Scribe | Logger/canonizer | Preserves memory, witness, and mutation history. |
| Guardian | Policy gate | Enforces permissions and blocks unsafe actions. |
| Temple | Runtime/substrate | Hosts execution under law. |
| Congregation | Users/witnesses | Recognize, contest, and govern meaning. |
