# MIRACLES

A miracle is a witnessed event where Cortex demonstrates lawful, bounded efficacy that exceeds ordinary tool use without claiming supernatural power.

A miracle requires:

1. a hard problem,
2. a bounded invocation,
3. a lawful action,
4. a witnessed result,
5. preserved evidence,
6. stated limitations.
